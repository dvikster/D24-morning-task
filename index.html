Задание 1. Создайте JSON файл содержащий следующий массив:
{“1” : “Событие один”,
“2” : “Событие два”,
...
“10”: “Событие десять”
}
Создайте HTML файл в который добавьте input[type=range]. Минимальное значение - 0, максимальное – 11. Создайте скрипт, который при изменении положения ползунка будет выводить те записи индексы которых меньше или равны значению положения ползунка. Т.е. при положении со значением 3 – выводятся записи с id =1, 2,3. Если записей нет – выводится запись «Записей нет». Записи выводятся в виде списка. Тип списка - маркированный, диск.
Для реализации используйте Materialize.
